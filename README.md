# Contact Manager App (MERN Stack)

This is a full-stack Contact Management application built using the MERN stack (MongoDB, Express, React, Node.js).

## Features
- **Add Contacts:** Add Name, Email, Phone, and optional messages.
- **View Contacts:** List all contacts in a modern UI.
- **Validation:** Server and Client-side validation for emails and required fields.
- **Search & Sort:** Live search and sorting functionality.
- **Delete:** Remove contacts easily.
- **Copy to Clipboard:** Click on Email/Phone to copy.

## Tech Stack
- **Frontend:** React.js, Tailwind CSS, Lucide Icons
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
